# Project Title: AI-Powered Payment Fraud Detection System

## Overview
A real-time AI system that detects fraudulent financial transactions using machine learning and anomaly detection.

## Features
- Real-time transaction risk scoring API
- Fraud detection using anomaly detection models
- Explainable AI
- Dashboard monitoring

## Tech Stack
- Frontend: Streamlit
- Backend: FastAPI
- AI: Scikit-learn
- Database: SQLite

## Constraints
- Low latency
- Secure data
